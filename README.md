# AdiARC Land Record Information Management System

What happens when you mix millions of handwritten land records, fragmented databases, and corrupted digital images? You get a massive data problem. AdiARC is a solo project designed and built from the ground up to automate, analyze, and audit this complex ecosystem in a single high performance web dashboard. It architected the frontend, backend database logic, and custom artificial intelligence pipelines to bring order to unstructured legacy data.

## Core Features

### Custom AI and OCR Engine

A fine tuned Vision Encoder Decoder machine learning model capable of accurately reading tricky Patwari handwriting off complex mutation documents. It bypasses the noise of standard documents to specifically read handwritten Number Shumar entries, complete with smart sequence indexing for multiple page Shajra attachments.

### Digital Image Forensics

Engineered custom extraction tools to rip through raw image files and extract hidden binary metadata. This tool bypasses standard EXIF readers and scans raw binary DNA to pull proprietary HMX and XMP tags injected by legacy systems, saving hundreds of hours of manual data entry.

### Wirasat Inheritance Engine

A full scale automated Islamic inheritance calculator. It dynamically maps visual family trees and calculates precise inheritance shares based on complex familial relationships.

### Deep Data Auditing and Database Automation

Integrated a comprehensive SQL and PowerShell toolkit directly into the user interface. This enables real time database health checks, instantly finding missing Khaiwat entries, syncing servers, and generating automated daily progress reports without ever leaving the web browser.

### Smart Inventory Management

A worker based scanning system for large scale file inventories. It uses background processing to index and verify massive directories of land record images, ensuring data integrity across large mauzas.
